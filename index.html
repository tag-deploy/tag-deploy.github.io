What is it?

Tag Deploy is a Continuous Integration tool for Google Tag Manager (GTM). In short it connects a GTM Tag to a file on Github. This allows for developers to use a workflow they are used to when writing code for tags. No longer do they need to login into GTM and copy/paste code into a textbox.
